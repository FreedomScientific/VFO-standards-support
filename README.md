# standards-support
Contains documentation for VFO software support of Web standards (HTML, CSS, and ARIA).

## Reporting issues

If you find an issue with Jaws or ZoomText support for HTML, CSS, or ARIA, please [file an issue](https://github.com/ThePacielloGroup/VFO-standards-support/issues?q=is%3Aopen) with the following information:
* Concise title that summarises the issue;
* Brief description of the issue;
* Name and version of Jaws/ZoomText;
* Name and version of the OS and browser;
* Test case that demonstrates the issue.

Note: please do not file general support issues here. To report general issues with Jaws, contact [Freedom Scientific](https://www.freedomscientific.com/About/ContactUs), and to report general issues with ZoomText, please contact [AI SQuared](https://www.aisquared.com/about-us/contact-us/).
